# OTP Generator and Verification UI

This is a simple OTP Generator and Verification interface built using HTML, CSS, and JavaScript. The project demonstrates DOM manipulation, event handling, and random number generation.

## Features

- Generate a random 4-digit OTP
- Enter OTP using 4 input fields
- Automatic cursor movement to next input field
- OTP verification system
- Displays result as:
  - Successful (Blue)
  - Wrong (Red)
- Input fields reset when generating a new OTP
- Random background color generator
- Clean UI using CSS Flexbox

## Technologies Used

- HTML
- CSS (Flexbox & Box Model)
- JavaScript

## JavaScript Concepts Used

- Math.random()
- Math.floor()
- DOM manipulation
- input event
- click event
- Conditional statements

## Project Purpose

This project was built to practice JavaScript DOM manipulation, event handling, and UI interaction.
